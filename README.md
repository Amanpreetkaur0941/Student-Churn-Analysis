# Student-Churn-Analysis
This project focuses on analyzing student engagement and churn patterns within an EdTech platform. By processing raw student data, I identified key factors contributing to dropouts and developed data-driven retention strategies to improve student lifetime value.


![Dashboard_Image](https://github.com/Amanpreetkaur0941/Student-Churn-Analysis/blob/165025033ca79ff8b79da170e433a47eee32fc91/Dashboard%20Screenshot.png)

## Technical Toolkit:

MS Excel: Data Cleaning, Pivot tables

## Getting Started

### How to use this project:
1. **Download** the `edtech_student_churn_dataset Final.xlsx` file.
2. **Open** the file and navigate to the `Clean_data` tab to view the processed dataset.
3. **Interact** with the Pivot Tables in the `Pivot_sheet` to filter churn rates by course.
4. **Read** the `Insights & Strategy` tab for the final business recommendations.
