# Student Churn Analysis – EdTech Platform
Student retention is one of the biggest profitability drivers in EdTech.  
This project analyzes student engagement behavior to identify churn patterns, key risk indicators, and opportunities to improve retention and lifetime value.

The objective was to transform raw student-level data into actionable business insights using structured Excel-based analysis.

## Business Problem

High student churn increases Customer Acquisition Cost (CAC) and reduces profitability.  

This analysis aims to:
- Identify when churn is highest
- Understand behavioral indicators of dropout
- Segment high-risk student groups
- Provide retention-focused recommendations
